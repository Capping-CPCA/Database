pgAdmin
=====================================
1. Download & install *Postgres 9.5*
2. Download & install *pgAdmin III*
3. Launch *pgAdmin*
4. Click on the `Add connection to a server` button
5. Fill in the following parameters:
 - *Name*: <choose_a_server_name>
 - *Host*: `10.11.12.21`
 - *Port*: `5432`
 - *Maintenance DB*: `postgres`
 - *Username*: `developer`
 - *Password*: <password_you_were_given>
6. Click `OK`
7. Double click on the server field on the side navigation
8. A prompt for the password should pop up if you haven't stored that password
